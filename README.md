# SharkeeHaptics
ESP8266 haptic system for VRChat

Apply Sharkee_Auto.cs to your VRChat Character in unity, it should find automatically some bones to work with, if not, manually assign them.
Boot up the sketch on an ESP8266 with a DRV2605L soldered to D1 and D2.
Boot up the Gui.py
Enjoy!
